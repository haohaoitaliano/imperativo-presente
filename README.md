# Imperativo presente

一个适合手机、iPad 和微信浏览器使用的意大利语 A2 在线复习测试，主题为 **Imperativo presente（现在时命令式）**。

## 内容与功能

测试包含 5 个部分：

1. Imperativo base（基础命令式）
2. Imperativo riflessivo（自反动词命令式）
3. Imperativo irregolare e pronomi（不规则命令式与代词）
4. Imperativo formale Lei（尊称 Lei 的命令式）
5. Completa la regola（完成规则总结）

学习者必须在每一部分全部答对后才能进入下一部分。答错后可以修改答案并再次检查；完成最后一部分后会显示完整的双语规则总结。

## 文件说明

- `index.html`：完整测试页面，包含 HTML、CSS 和 JavaScript
- `README.md`：项目说明与部署方法
- `.nojekyll`：让 GitHub Pages 直接发布静态文件

项目不使用框架、外部字体、外部图片或外部 JavaScript，也不需要后端和登录。

## 本地预览

可以直接用浏览器打开 `index.html`。也可以在项目目录运行一个本地静态文件服务器，例如：

```bash
python3 -m http.server 8000
```

然后打开 `http://localhost:8000`。

## GitHub Pages

在 GitHub 仓库中依次打开：

`Settings → Pages → Deploy from a branch → main → /(root)`

保存后即可通过 GitHub Pages 访问。
